## Dependencies

* NodeJS (https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-an-ubuntu-14-04-server)
* NPM

*Hint: Node binary is called `nodejs` for stock apt repo.*

## Install

```sh
$ cd app
$ npm install
```

## Start server (exposes port 3000)

```sh
$ cd app
$ node index.js
```
